# Kliiko
Tested on: Node version 4.2.1 LTS, NPM version 3.3.8

## Dependencies

NodeJs 4.2.x

NPM 3.3.x

MySQL 5.7

Redis 3.0.5

[ImageMagick](http://www.imagemagick.org/)

## Set up Project

```sh
 cd project_path
 npm install
 npm install mocha -g
 npm install gulp -g
```

### Database ORM

Use [Sequelize](http://docs.sequelizejs.com/en/latest/)
